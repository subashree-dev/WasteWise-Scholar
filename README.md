# WasteWise Scholar

AI-Powered Sustainable Study Assistant

## Project Overview
WasteWise Scholar helps students reduce unnecessary paper printing by providing:
- Smart document summaries
- Topic extraction
- Print recommendations
- Sustainability impact estimates

## SDG Alignment
- SDG 12: Responsible Consumption and Production
- SDG 4: Quality Education
- SDG 13: Climate Action
